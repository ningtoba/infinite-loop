# Security Policy

## Supported Versions

Only the latest release of this project is actively maintained with security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < latest| :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by:

1. **Opening a GitHub Issue** — tag it with `security` for visibility.
2. **Emailing** security@example.com — we aim to respond within **48 hours**.

We do not require a PGP key at this time. Please include a description of the issue, steps to reproduce, and any relevant impact analysis.

You should receive an initial acknowledgment within 48 hours. We will work with you to understand the scope, develop a fix, and coordinate disclosure.
